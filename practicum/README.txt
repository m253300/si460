Run the program with "python3 koutrakos.py"

What it does: 
Terrain creator. Press 'enter' and the terminal will prompt you to type numbers specifying row, column and height in the terrain matrix. Press 'enter' after each number input and it will prompt you for the next number. After entering height and pressing 'enter', it will modify that part of the matrix to that specified height.

You can also increase the size of the matrix my using the 'up' arrow and decrease it with the 'down' arrow. This will reset the matrix to flat. 

You can also zoom in and out using the mouse scroll wheel and reorient the world by clicking and dragging with the mouse.
